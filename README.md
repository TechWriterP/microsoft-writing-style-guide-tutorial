# Microsoft Writing Style Guide Tutorial

A documentation practice project based on Microsoft Writing Style Guide principles.

Live site: [https://techwriterp.github.io/msg-tutorial/](https://techwriterp.github.io/msg-tutorial/)

## Purpose

This project demonstrates technical writing practice around style, consistency, clarity, and user-focused documentation. It shows how editorial guidance can be converted into structured learning content.

## Audience

- Technical writers practicing style-guide based writing
- Documentation teams interested in editorial consistency
- Recruiters and hiring managers reviewing writing and documentation samples

## Documentation Skills Demonstrated

- Style-guide interpretation
- Concise technical writing
- Editorial consistency
- Tutorial-style documentation
- Static documentation publishing

## What to Review

Review the live site and repository content for examples of structured tutorial writing and style-guide based documentation practice.
